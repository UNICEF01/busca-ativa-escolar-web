Coordenador Operacional

Agora, informe os dados do Coordenador Operacional. Ele é o responsável por:

  • Cadastrar os supervisores institucionais, os técnicos verificadores e os agentes comunitários;
  • Estabelecer os parâmetros do sistema de acordo com a realidade do município;
  • Acessar as estatísticas gerais da cidade;
  • Definir, junto com o gestor político, as estratégias para realizar o projeto no município;
  • Coordenar a formação das equipes participantes.
